> # 추가 사항

## 다이얼 회전 기능 및 커스텀 강화

- 다이얼 회전 동작을 더 다양한 방식으로 활용할 수 있도록 확장했습니다.
- 마우스 이동 기능에 후속 액션을 추가해, 연속 동작을 더 편하게 설정할 수 있습니다.
- 마우스 버튼 클릭 동작을 사용자 환경에 맞게 커스터마이징할 수 있습니다.

## 다이얼 스크롤 커스터마이징 지원

- 다이얼 스크롤 실행 방식을 사용자 목적에 맞게 설정할 수 있도록 개선했습니다.

> # 수정 사항

## Windows 커스텀 폰트 인식 문제 해결

- Windows 환경에서 커스텀 폰트를 정상적으로 인식하지 못하던 문제를 수정했습니다.

## Windows 키보드 키 해제(Release) 동작 안정화

- Windows에서 키보드 키 해제(Release) 메시지가 정확히 전달되지 않던 문제를 해결했습니다.

## Adobe Premiere Pro 기본 수치 값 조정

- Adobe Premiere Pro에서 기본 수치 조작 값을 `0.1`로 변경했습니다.
